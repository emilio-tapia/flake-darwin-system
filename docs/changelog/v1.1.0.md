# Changelog — v1.1.0

**Fecha:** 2026-03-26
**Tipo:** chore
**Alcance:** general

## Descripción

Unificación de inputs a 25.05 con Determinate Nix, correcciones de home-manager y PostgreSQL, actualización y verificación de paquetes Homebrew en ambos hosts, y documentación de skills.

## Cambios realizados

1. Se unificaron todos los inputs del flake a nixpkgs 25.05, nix-darwin 25.05 y home-manager release-25.05
2. Se integró `determinate.darwinModules.default` en ambas configuraciones de host
3. Se dinamizó el usuario de PostgreSQL vía `specialArgs` eliminando hardcoding de usernames
4. Se comentaron paquetes no disponibles en nixpkgs 25.05 con explicación
5. Se corrigió `zsh.dotDir` a ruta relativa y se añadieron guardas `optionalAttrs` para pgcli/jqp
6. Se limpiaron templates en home-manager
7. Se corrigieron configs de Homebrew en ambos hosts (nombres de casks, taps)
8. Se habilitaron brews verificados: llmfit, models, whosthere, mole, snitch, bookokrat, dnspyre, cronboard, gittype
9. Se sincronizaron brews entre m4Pro y macbookPro
10. Se habilitaron casks en macbookPro: cursor, figma, losslesscut, handbrake-app
11. Se deshabilitó docker-desktop en macbookPro
12. Se añadió documentación de code review analysis y Claude skills

## Archivos modificados

- `flake.nix` — inputs unificados a 25.05, integración Determinate
- `flake.lock` — actualización de lock file
- `darwinModules/development/devTools.nix` — usuario dinámico de PostgreSQL vía specialArgs
- `home_manager/emilio_m4Pro.nix` — dotDir relativo, guardas condicionales para pgcli/jqp
- `home_manager/admin_macbookPro.nix` — dotDir relativo, guardas condicionales para pgcli/jqp
- `hosts/m4Pro/homeBrew.nix` — brews verificados habilitados, corrección de nombres de taps
- `hosts/macbookPro/homeBrew.nix` — sincronización de brews, casks habilitados, docker-desktop deshabilitado
- `docs/code-review.md` — análisis de code review
- `.claude/skills/` — skills de Claude configurados

## Impacto para el usuario

Ambos hosts usan inputs estables 25.05 con Determinate Nix. Nuevas herramientas CLI disponibles vía Homebrew (llmfit, models, mole, snitch, etc.). MacBook Pro Intel gana acceso a cursor, figma y herramientas de video. Configuración más robusta con guardas condicionales entre versiones.
