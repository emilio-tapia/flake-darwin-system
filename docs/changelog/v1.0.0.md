# Changelog — v1.0.0

**Fecha:** 2026-03-22
**Tipo:** feat
**Alcance:** general

## Descripción

Release inicial del flake nix-darwin. Configuración completa de dos hosts macOS (M4 Pro Apple Silicon y MacBook Pro 2015 Intel), home-manager integrado, Homebrew gestionado declarativamente, y devenv para shells de desarrollo.

## Cambios realizados

1. Se creó el flake inicial para MacBook Comtelec (Intel x86_64-darwin)
2. Se añadió configuración completa del host M4 Pro (aarch64-darwin): darwin-configuration, dev-tools, homebrew
3. Se configuró home-manager como módulo de nix-darwin para ambos hosts (emilio_m4Pro, admin_macbookPro)
4. Se creó `mkHomeConfiguration` helper function en flake.nix
5. Se integró nix-homebrew para gestión declarativa de Homebrew con Rosetta en Apple Silicon
6. Se instalaron herramientas de terminal: lazyvim, atuin, hurl, eza, hyperfine, ripgrep, fd, jq, btop, tmux
7. Se configuraron herramientas de desarrollo: graphviz, plantuml, PostgreSQL, pgcli
8. Se añadieron paquetes vía Homebrew: n8n, oxker, doppler, poppler, losslesscut, handbrake
9. Se instaló Claude Code vía Homebrew
10. Se configuró p10k como zsh theme
11. Se separaron devtools por plataforma específica
12. Se reorganizó Homebrew para Apple Silicon con brewPrefix
13. Se configuró devenv con perfiles de desarrollo
14. Se refactorizó el flake principal con estructura modular (darwinModules, hosts, home_manager, home_managerModules)
15. Se migró Docker Desktop a Homebrew (en lugar de nix)
16. Se habilitó system.primaryUser
17. Se movieron home-manager packages de system-wide a per-user
18. Se hicieron jqp y pgcli condicionales por compatibilidad de versión HM
19. Se actualizaron inputs múltiples veces hasta estabilizar en 25.05
20. Se corrigió dotDir a ruta relativa en configuración zsh
21. Se inicializó sistema de versionamiento con archivo VERSION

## Impacto para el usuario

Sistema completo de gestión declarativa para dos Macs. Todos los paquetes, herramientas de desarrollo, aplicaciones GUI y configuraciones del sistema se gestionan desde un único flake. `darwin-rebuild switch` aplica todo incluyendo home-manager y Homebrew.
